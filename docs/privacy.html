Privacy Policy

Last updated: 13 June 2026

This page describes how PrePrompt (the open-source tool at github.com/Preprompt-ai/preprompt) and the preprompt.org website handle your data. It is plain-English and aims to be accurate, not exhaustive. If anything here disagrees with the code in the public repository, treat the code as authoritative and please open an issue.

1. PrePrompt on your machine

PrePrompt is software that runs on your computer. It stores the prompts it processes, the rewrites it produces, your Anthropic API key, and accumulated technology-stack signals in ~/.preprompt/ on your local disk. These files are restricted to your user account (chmod 600). They are not transmitted to any server we operate.

2. Calls to Anthropic

When the classifier decides a prompt needs rewriting, PrePrompt sends that prompt — and recent conversation history — directly to Anthropic's API using your own API key. We do not see, log, or proxy this traffic. Anthropic's handling is governed by their privacy policy.

3. Telemetry (off by default)

PrePrompt can optionally send a small set of metadata events to PostHog — currently prompt_processed, enrichment_accepted, enrichment_rejected, install_completed, and secret_detected. These events never include prompt text. They contain only a route label (pass / enrich / clarify), a length, a score, and (for the secret_detected event) a count.

Telemetry is disabled unless you set POSTHOG_API_KEY in your environment. We will never enable telemetry without action on your part.

4. Secret scanning

Before any prompt is sent to the Anthropic API, PrePrompt's secret scanner inspects it for patterns that look like API keys, JWT tokens, AWS keys, GitHub tokens, and similar credentials. If a match is found, the prompt is not sent to Anthropic, and the placeholder [SECRET DETECTED - NOT LOGGED] is written to local history in place of the real content.

5. The preprompt.org website

This website is a static page hosted on a CDN. It does not require an account to browse. The "live stats" widget displays aggregate counts only. We use Plausible-style privacy-respecting analytics (no cookies, no personal identifiers).

6. The hosted Pro tier

The hosted optimizer (when launched) will receive prompts on our servers in place of direct calls to Anthropic. We will publish a separate, more detailed privacy notice for that service before launch — including retention periods, the regions in which data is processed, and the sub-processors we use. Until that notice exists, the hosted tier is not available.

7. Children

PrePrompt is not directed at children under 13. We do not knowingly collect data from anyone under 13.

8. Your rights

Everything PrePrompt knows about you lives in ~/.preprompt/ on your own machine. Delete that directory and you've deleted everything. For any data the website processes, write to privacy@preprompt.org and we'll respond within 30 days.

9. Changes

If we change this policy, we will update the date above and surface a note in the next release's changelog. Material changes will be announced on the website and the GitHub repository.
